Runa Plug Tail

Summary

This package is a latex bunny suit for the Runa model by Torinyan. You must own the Runa base model to use it.

Installation

  1. Unpack the dependencies, in order.
  2. Unpack Runa_Bunny_Suit.unitypackage.
  3. Drag Assets/Kemono_Io/Runa_Bunny_Suit.prefab into the Runa_Source object in the Hierarchy panel.
  4. Set the Runa_Bunny_Suit's anchor override.
    1. Select the Runa_Bunny_Suit mesh in the Hierarchy panel.
    2. Drag the hips bone into the Anchor Override in the Inspector panel.
  5. Unpack the Runa_Bunny_Suit prefab in the Hierarchy panel.
  6. Constrain the Runa_Bunny_Suit's armature to the Runa base model's armature.
    1. Fully expand the Runa_Bunny_Suit object's armature in the Hierarchy panel.
    2. Select one of the bones, such as hips.
    3. Drag the identically-named bone from the Runa base model's armature to the Parent Constraint field in the Inspector panel.
    4. Click the Activate button.
    5. Repeat the last three steps for each bone in the armature.
  7. Optionally, the transparency of the bunny suit can be altered.
    1. Select the Runa_Bunny_Suit object in the Hierarchy panel.
    2. Alter the Alpha Mod property in the Inspector Panel to a value between 0 and -1, where 0 is full opacity and -1 is full transparency.
